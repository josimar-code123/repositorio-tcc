# Repositório de Trabalhos de Conclusão de Curso (TCCs)

Bem-vindo ao Repositório de TCCs! Este espaço foi criado para que os alunos possam compartilhar e acessar trabalhos de conclusão de curso de diversas áreas e cursos.

## Proposta do Repositório

O objetivo deste repositório é criar uma comunidade onde os alunos possam compartilhar seus TCCs, proporcionando uma oportunidade para aprendizado, inspiração e colaboração. Aqui, você encontrará uma variedade de trabalhos de diferentes áreas e cursos, representando a diversidade acadêmica.

## Como Contribuir

Se você é um aluno e deseja contribuir com seu próprio TCC, siga estas etapas simples:

1. **Faça um Fork do Repositório:**
   - Clique no botão "Fork" no canto superior direito desta página para criar uma cópia do repositório na sua conta.

2. **Clone o Repositório Forked:**
   - Abra o terminal no seu computador.
   - Execute o comando `git clone https://github.com/seunome/repo-tcc.git`, substituindo "seunome" pelo seu nome de usuário no GitHub.

3. **Adicione Seu TCC:**
   - Coloque seu TCC na pasta apropriada, seguindo a estrutura de pastas existente.
   - Se sua área, curso ou ano não estiver representado, sinta-se à vontade para criar pastas adicionais conforme necessário.

4. **Commit e Push:**
   - Execute os comandos:
     ```bash
     git add .
     git commit -m "Adicionando meu TCC sobre [assunto]"
     git push origin master
     ```

5. **Crie um Pull Request (PR):**
   - Volte ao repositório no GitHub.
   - Clique no botão "Compare & pull request" ao lado do seu último commit.
   - Preencha as informações do PR e clique em "Create pull request".

6. **Aguarde a Revisão:**
   - Aguarde a revisão do mantenedor do repositório. Se necessário, esteja preparado para realizar ajustes conforme solicitado.

Obrigado por contribuir para o nosso repositório de TCCs! Sua participação é valiosa para construir uma comunidade acadêmica diversificada e enriquecedora.

## Diretrizes de Contribuição

Para garantir uma experiência positiva para todos os usuários, solicitamos que siga estas diretrizes ao contribuir:
- Certifique-se de que seu TCC está em conformidade com as leis de direitos autorais.
- Evite incluir informações pessoais sensíveis nos documentos.
- Utilize pastas apropriadas para categorizar seu TCC.

Agradecemos sua colaboração e esperamos que este repositório seja uma fonte valiosa de conhecimento e inspiração!
